Projetos

Projeto Hamburgueria

Uma aplicação web para gerenciamento de pedidos e cardápio de uma hamburgueria local. Desenvolvido com HTML, CSS e JavaScript, este projeto apresenta uma interface amigável para clientes fazerem pedidos online.

MapSP

Uma aplicação para mapeamento de treinos de corrida e ciclismo. Desenvolvido com Leaflet.js e JavaScript, este projeto permite aos usuários rastrearem seus treinos e explorarem novas rotas diretamente no mapa.

Solução para Oceanos

O site da BlueSphere oferece uma visão inovadora e tecnológica para a preservação dos oceanos. Utilizando soluções avançadas de IoT (Internet das Coisas), a BlueSphere monitora e melhora a qualidade da água em tempo real. Com sensores estratégicos e análise de dados na nuvem, a empresa fornece insights cruciais para a conservação marinha. Seu compromisso com um futuro sustentável é evidente em cada aspecto do seu trabalho, buscando proteger e preservar os ecossistemas oceânicos para as gerações futuras.

BikeCity

Uma plataforma online para compartilhamento de rotas e informações sobre ciclismo urbano. Desenvolvido como parte de um projeto colaborativo, este projeto apresenta integração com Node.js para fornecer uma experiência completa aos usuários.
